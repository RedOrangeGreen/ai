AI Bare Metal Setup (C++) on Ubuntu Live USB

Follow these steps to set up and run an AI model using llama.cpp on Ubuntu 24.04.3 LTS Live USB.

            
cd /home/ubuntu/Public
sudo apt install -y git build-essential cmake libcurl4-openssl-dev libssl-dev
git clone https://github.com/ggml-org/llama.cpp
cd llama.cpp
cmake -B build
cmake --build build --config Release
./build/bin/llama-cli -hf ggml-org/gemma-3-1b-it-GGUF
