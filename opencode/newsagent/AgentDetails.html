Agent Details - Price Agent

Overview

This project demonstrates the integration of llama.cpp and agent.cpp to create a local AI agent that can fetch international news headlines from BBC RSS feed using a locally-running Large Language Model (LLM).

Components Built:

  • COMPLETED llama.cpp - LLM inference engine (C/C++)
  • COMPLETED agent.cpp - Agent framework for local agents (C++)
  • COMPLETED Gemma 4 E4B - GGUF quantized model (4.96 GB)
  • COMPLETED News Agent - Custom agent that fetches BBC news headlines

What Was Built

1. llama.cpp

llama.cpp is a C/C++ library for LLM inference with minimal setup and state-of-the-art performance. It enables running LLMs locally on CPU and GPU.

  • Source: https://github.com/ggml-org/llama.cpp
  • Version: Latest master branch (commit 660b1b4)
  • Build type: Release with OpenMP support
  • Output: Shared libraries (libllama.so, libllama-common.so, etc.)

2. Gemma 4 E4B GGUF Model

The smallest Gemma 4 model quantized to Q4_K_M format for efficient local inference.

  • Model: google/gemma-4-E4B-it
  • Format: GGUF v3, Q4_K_M quantization
  • Size: 4.96 GB
  • Source: https://huggingface.co/ggml-org/gemma-4-E4B-it-GGUF
  • Context: 131,072 tokens
  • Parameters: ~8B active parameters (MoE architecture)

3. agent.cpp

Mozilla's agent.cpp provides building blocks for creating local agents that use llama.cpp for LLM inference. It supports tool calling, callbacks, and agent loops.

  • Source: https://github.com/mozilla-ai/agent.cpp
  • Version: Latest main branch
  • Integration: Bundled with compatible llama.cpp (submodule)
  • Features: Tool support, agent loops, callbacks, memory management

4. News Agent Example

A custom agent that demonstrates tool usage - it fetches 5 current international news headlines from BBC World News RSS feed. Runs forever with 2-second delays.

  • Location: /home/ubuntu/Downloads/agent.cpp-main/examples/price-agent/
  • Tool: get_news - Fetches 5 headlines from BBC RSS feed
  • Loop: Runs forever with 2-second delay between headline fetches
  • Output: Agent uses LLM to interpret user request and call the tool
  • RSS Parser: Uses Python script to fetch and parse BBC RSS feed

Agent Functionality

What the Agent Does (Runs Forever with 2-Second Delays)

  1. Loop: Runs forever, each time fetching 5 current news headlines
  2. 2-Second Delay: Waits 2 seconds between each news check
  3. RSS Fetch: Python script fetches BBC World News RSS feed
  4. LLM Processing: The Gemma 4 model processes the prompt and decides to use the get_news tool
  5. Tool Execution: The NewsTool::execute() returns headlines in JSON format
  6. Response Generation: The LLM formats the tool result into a human-readable response
  7. Model Reset: KV cache is reset between runs for fresh context

Tool Definition

Tool Name: get_news
Description: Get 5 current international news headlines from BBC World News RSS feed.
Parameters: None required
Returns: JSON with headlines array, source, and timestamp

Example return value:
{
"headlines": ["headline1", "headline2", "headline3", "headline4", "headline5"],
"source": "BBC World News RSS",
"timestamp": "Thu Apr 30 16:45:00 2026",
"count": 5
}

Code Highlights

  • NewsTool class: Inherits from agent_cpp::Tool and implements execute() method
  • RSS Feed: Uses Python script to fetch BBC World News RSS feed
  • Infinite Loop: Agent runs forever with 2-second delays between each news grab
  • Model Reset: Model::reset() clears KV cache between runs for fresh context
  • Agent loop: Uses agent_cpp::Agent::run_loop() to process messages and handle tool calls
  • Model config: 2048 context length, temperature 0.7, top_p 0.9, top_k 40
