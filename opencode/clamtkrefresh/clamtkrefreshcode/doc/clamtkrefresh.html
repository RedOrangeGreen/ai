ClamTK Refresh

Modern GUI Redesign for ClamTk Virus Scanner

Version 7.0 Refresh

📋 Overview

ClamTK Refresh is a complete modernization of the ClamTk virus scanner GUI. Built on the original ClamTk codebase by Dave M, this refresh transforms the user interface with a contemporary design inspired by modern design principles.

The original ClamTk uses a traditional icon-view based layout. ClamTK Refresh reimagines this with a clean, card-based design that improves usability and visual hierarchy.

✨ Key Changes Made

1. Modern Visual Design

Card-Based Layout

Replaced icon views with modern, clickable cards that have hover effects and clear visual boundaries

CSS Styling

Added comprehensive CSS styling with custom properties for consistent theming

Color Palette

Modern color scheme with #007aff blue, clean whites, and subtle grays

Typography

Improved font weights, sizes, and hierarchy for better readability

2. Improved Layout Structure

Original Design

  • Icon views with small 24px icons
  • Vertical stacking of all elements
  • Basic separators between sections
  • Limited visual hierarchy
  • Standard GTK theme

Refresh Design

  • Grid layout with compact cards (150x110px)
  • Organized sections: Scan, Protection, Settings
  • Compact hero section (80px height)
  • All options visible without scrolling (1100x800 window)
  • Custom CSS with modern aesthetics

3. Technical Improvements

  • CSS Provider Integration: Added Gtk3::CssProvider for custom styling
  • Grid Layout: Replaced linear icon views with Gtk3::Grid for better organization
  • Modern Cards: Created create_modern_action_card() function for consistent card-based buttons
  • Section Headers: Added create_modern_section() for organized content grouping
  • Optimized Icons: Set icon size to 36px for compact yet visible cards
  • Larger Window: Window default size increased to 1100x800 to show all options without scrolling
  • Compact Cards: Reduced card size to 150x110px to fit all options in view

🔧 Files Modified

Primary Changes

  • lib/GUI.pm - Complete redesign of the graphical user interface
  • clamtkrefresh - New executable script pointing to the refreshed GUI

Lib/GUI.pm Changes

Functions Added:

  • create_modern_section() - Creates section headers with title and subtitle
  • create_modern_action_card() - Creates styled card buttons with icons

Functions Modified:

  • start_gui() - Added CSS provider, changed window size, new header bar
  • add_default_view() - Complete rewrite with card-based layout

🚀 How to Build

Prerequisites: Perl 5, Gtk3 Perl module, ClamAV installed

Method 1: Direct Run (Recommended)

cd /home/ubuntu/Downloads
./clamtkrefresh

This launcher script automatically sets the correct Perl library path to use the modern GUI.

Method 2: Run from Source Directory

cd /home/ubuntu/Downloads/clamtk-master
perl clamtkrefresh

Method 3: System-Wide Installation

cd /home/ubuntu/Downloads/clamtk-master
./build.sh
sudo cp -r build/* /

After installation, you can run directly:

clamtkrefresh

Build Script

Use the provided build script to create the build directory structure:

cd /home/ubuntu/Downloads/clamtk-master
./build.sh

This will:

  • Create the build directory structure
  • Copy the executable to build/usr/local/bin/
  • Copy Perl modules to build/usr/share/perl5/ClamTk/
  • Prepare desktop file and icon

✓ GUI Changes Verified! The modern GUI is loaded from /home/ubuntu/Downloads/clamtk-master/lib/ClamTk/GUI.pm

Running Without Installation

cd /home/ubuntu/Downloads

This launcher script automatically sets the correct Perl library path to ensure the modern GUI is used.

Running from Source Directory

cd /home/ubuntu/Downloads/clamtk-master

Running After Installation

clamtkrefresh

Command-Line Scanning

clamtkrefresh /path/to/file
clamtkrefresh /path/to/directory
Note: ClamTK Refresh requires ClamAV to be installed on your system. Install it with: 
sudo apt install clamav clamav-daemon  # Debian/Ubuntu
sudo dnf install clamav clamav-update    # Fedora
sudo yum install clamav clamav-update    # CentOS/RHEL

Window (340x400 - requires scrolling)
├── HeaderBar

└── Main Box (vertical)
    ├── Configuration Section
    │   └── IconView (4 columns)
    ├── History Section  
    │   └── IconView (3 columns)
    ├── Updates Section
    │   └── IconView (3 columns)
    └── Analysis Section
        └── IconView (3 columns)

Window (1100x800 - no scrolling needed)
├── HeaderBar (with subtitle & menu)
├── InfoBar
└── ScrolledWindow
    └── Main Box (vertical, 12px spacing)
        ├── Hero Card (title + subtitle, 80px height)
        ├── Scan Section
        │   └── Grid (3 columns, 8px spacing)
        │       ├── Scan File Card (150x110px)
        │       ├── Scan Directory Card (150x110px)
        │       └── Analysis Card (150x110px)
        ├── Protection Section
        │   └── Grid (3 columns, 8px spacing)
        │       ├── History Card (150x110px)
        │       ├── Quarantine Card (150x110px)
        │       └── Whitelist Card (150x110px)
        └── Settings Section
            └── Grid (4-5 columns, 8px spacing)
                ├── Preferences Card (150x110px)
                ├── Update Card (150x110px)
                ├── Network Card (150x110px)

🎨 CSS Styling

The refresh includes a complete CSS stylesheet with:

  • Custom Properties: Defined colors for easy theming
  • Header Bar: Gradient background with subtle shadow
  • Buttons: Rounded corners, hover effects, shadow
  • Cards: White background, border, rounded corners, shadow
  • Color Scheme: Apple-inspired with #007aff accent color

Key CSS Variables:

@define-color bg_color #f5f5f7;
@define-color card_color #ffffff;
@define-color accent_color #007aff;
@define-color accent_hover #0056b3;
@define-color text_primary #1d1d1f;
@define-color text_secondary #86868b;

📦 Distribution

Executable Location

After building, the executable is located at:

/home/ubuntu/Downloads/clamtk-master/clamtkrefresh
/home/ubuntu/Downloads/clamtkrefresh (copy in Downloads root)

Perl Modules

The required Perl modules are in:

/home/ubuntu/Downloads/clamtk-master/lib/*.pm

Packaging

To create a distributable package:

cd /home/ubuntu/Downloads/clamtk-master
tar czf clamtkrefresh-v7.0.tar.gz build/

🐛 Known Issues & Limitations

  • Requires Perl Gtk3 modules: sudo apt install libgtk3-perl
  • CSS styling may vary depending on system GTK theme
  • Some icon names may not exist on all systems (fallback handled by ClamTk::Icons)
  • Original ClamTk is no longer maintained (see README.md)
