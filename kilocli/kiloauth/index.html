Project Overview

kiloauth.sh is an AI-generated automation script that builds a functional multi-account TOTP desktop application. The final output is a compiled binary named MultiTOTPViewer.

Build Environment

Ubuntu Linux 24.04.4 LTS (Live USB)
Standard Bash Shell

Autonomous AI Build Process

Summary: Create shell script (using AI chatbots to help with this) then run shell script to create App.

The shell script downloads Kilo CLI and all required/associated development tools then generates and builds the code into a binary App.

Step 1 – Human Intent & Instruction Synthesis:
Define high-level product goals and convert these into a shell script (kiloauth.sh) containing structured engineering plans and automation logic, using AI chatbots to help create the script..

Step 2 – Script Execution:
Invoke the shell script locally using:
bash ./kiloauth.sh

Step 3 – Code Generation & Assembly:
Kilo CLI executes the build instructions and generates the working application, a compiled Linux binary called MultiTOTPViewer

What This Demonstration Illustrates

This project demonstrates a complete concept-to-binary pipeline where:

  • A human defines intent and converts intent into engineering instructions, using AI chatbots to help.
  • A CLI automation tool builds the software autonomously
  • The final result is a functioning application
