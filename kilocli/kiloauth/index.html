Project Overview

kiloauth.sh is an AI-generated automation script that builds a fully functional multi-account TOTP desktop application. The final output is a compiled binary named MultiTOTPViewer.

Autonomous AI Build Process

Summary: Use ChatGPT to create script file which 'invokes Kilo CLI to generate and build source code' then run script file to create App

Step 1 – Human Intent:
High-level product goals were defined using ChatGPT .

Step 2 – Instruction Synthesis:
ChatGPT transformed these goals into a shell script (kiloauth.sh) containing structured engineering plans and automation logic.

Step 3 – Script Execution:
The shell script was invoked locally using:
bash ./kiloauth.sh

Step 4 – Code Generation & Assembly:
Kilo CLI executed the build instructions and generated the working application.

Final Result:
A ready-to-run, compiled Linux binary was generated: MultiTOTPViewer
