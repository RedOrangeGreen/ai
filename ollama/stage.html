Ollama: Ubuntu Version, Free, Anonymous, Open Source, No API Keys Required, Runs 100% On Local Machine

The following steps show how to use Docker on Ubuntu to install Ollama from dockerhub, use then uninstall it. By default, Ollama provides a 'reasonably quick' command line AI chatbot with answers generated using the Meta Llama 3 (8B) model.

What's Docker? Docker's a platform that allows developers to easily create, package, and run software 'images' in 'containers', ensuring they work consistently across different environments. The format "username/imagename" is used to identify Docker images on 'dockerhub', "ollama" is used for both the username and imagename by Ollama.

'For a 5 year old' definition of 'Docker'..."Docker is like a special toy box that helps keep computer programs organized, a Docker image is a packed box of everything needed to play a game, and a Docker container is when you take that box out to play with it!".

Install: Add Docker, Start Docker, Download And Run The Ollama Image (ollama/ollama) In A Container (aidoings)

  • sudo apt install docker.io
  • sudo systemctl start docker
  • sudo docker run -d -v ollama:/root/.ollama -p 11434:11434 --name aidoings ollama/ollama

Use: Invoke llama3 In The Container Image (First Invoke Takes Minutes As It Downloads The AI Model)

  • sudo docker exec -it aidoings ollama run llama3
  • What is a dinosaur and are any alive today?
  • /bye
  • sudo docker exec -it aidoings ollama run llama3
  • Show me hello world in C++
  • Add a class called father and two classes called son and daughter, both of which inherit from father
  • /bye

Uninstall: Stop And Remove Container (aidoings), Remove Image (ollama/ollama), Stop Docker, Remove Docker

  • sudo docker stop aidoings
  • sudo docker rm aidoings
  • sudo docker rmi ollama/ollama
  • sudo systemctl stop docker.service
  • sudo systemctl stop docker.socket
  • sudo apt-get purge --auto-remove docker.io
